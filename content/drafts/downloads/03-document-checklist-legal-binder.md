## Recommended Binder Sections

Check off each section as you create it. Write the page range or number of documents in each section.

### Section 1 — Case Summary (Cover Sheet)

- [ ] One-page summary: who you are, case number, court, sentence, what you are requesting
- [ ] Table of contents listing each section below

### Section 2 — Court & Sentencing Records

- [ ] Judgment / sentencing order

- [ ] Pages: _____ to _____

### Section 3 — Educational Achievements

- [ ] Program completion certificates
- [ ] Transcripts, diplomas, GED
- [ ] Teacher or program coordinator letters

---

## Quality Check (Before Submitting)

- [ ] Every section has a tab or divider label
- [ ] No duplicate documents in multiple sections unless cross-referenced
- [ ] Every document is readable (no dark photocopies)
- [ ] Sensitive information is included only when relevant
- [ ] A friend or advocate has reviewed for clarity (if possible)
